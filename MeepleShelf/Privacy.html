What is Meeple Shelf? ¿Qué es Meeple Shelf?

Meeple Shelf is an Android app for board game enthusiasts. It connects to a user's public BoardGameGeek collection and lets them browse, search and sort their games by name, year, BGG rank, rating, player count or playing time. From each game, users can jump directly to its rulebooks, FAQs and player aids in their preferred language.

Meeple Shelf es una app Android para aficionados a los juegos de mesa. Se conecta a la colección pública del usuario en BoardGameGeek y permite explorarla, buscarla y ordenarla por nombre, año, ranking, nota, número de jugadores o duración. Desde cada juego se accede directamente a sus manuales, FAQs y ayudas de juego en el idioma preferido.

Data source & attribution Fuente de datos y atribución

Game data is provided by the BoardGameGeek XML API and used in accordance with the BGG XML API Terms of Use. BoardGameGeek is credited as the data source within the app, and every game links back to its page on boardgamegeek.com.
